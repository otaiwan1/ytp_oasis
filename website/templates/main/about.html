About OASIS

Learn about the technology and research behind the system.

Project Overview

OASIS (Orthodontic AI Similarity Intelligent System) is a research project developed under the YTP 2025 (Young Turing Program). The goal is to assist orthodontists by providing an AI-powered tool that can search for similar dental cases from a database of 3D scans.

When a dentist encounters a new patient, they can upload the patient's 3D dental scan (STL file) and our system will find the most similar historical cases. This helps dentists reference past treatment plans and outcomes for similar malocclusion patterns.

Technical Architecture

SimCLR Framework

Self-supervised contrastive learning to create meaningful dental scan embeddings without requiring labeled data.

EdgeConv (DGCNN)

Dynamic graph neural network that captures local geometric structures in 3D point clouds through k-nearest neighbor graphs.

4-Layer Architecture

3 → 64 → 64 → 128 → 256 channels, aggregated into a 512-dimensional embedding for robust similarity comparison.

NT-Xent Loss

Normalized temperature-scaled cross entropy loss for effective contrastive learning with temperature τ = 0.1.

Data Pipeline

1
STL Collection

3D dental scan files collected from orthodontic practices via OrthoCAD exports.

2
Point Cloud Extraction

Mesh surfaces sampled to 4096 normalized 3D points with centroid alignment and unit scaling.

3
Data Augmentation

Rotation, scaling, jittering, and cuboid cutout for robust contrastive pair generation.

4
Multi-View Rendering

6-view rendering (front, back, top, bottom, left, right) using Open3D for visualization.

5
Feedback Loop

Dentist feedback is collected to fine-tune the model for improved results over time.
